Local-first · Android

Download. Summarize. Share.

BabakCast lets you grab YouTube videos and turn transcripts into summaries using your AI provider. No backend, no accounts, no tracking.

Get the app View source

What it does

  • ↓

    YouTube download

    Paste a URL, get shareable video files. Auto-split at 16 MB so you can share anywhere.

  • ∑

    Transcript → summary

    Extract captions and summarize with your chosen model. Bullets, paragraph, or TL;DR.

  • ⊕

    Bring your own AI

    OpenAI, Azure, Anthropic, Gemini, OpenRouter. Set API key and model in Settings.

  • ◆

    Stays on device

    Everything runs locally. API keys stored with Android EncryptedSharedPreferences.

BabakCast is a personal-use tool. It does not ship with API keys, ads, analytics, or accounts. You control your data and your AI provider.

Install

From source

git clone https://github.com/cocodedk/BabakCast.git
cd BabakCast

Open in Android Studio and run (API 26+).

APK

Download the latest release from GitHub and install on your device.

Releases

Requires Android 8.0+. For summarization, add an API key in Settings (OpenAI, Anthropic, Gemini, OpenRouter, or Azure OpenAI).
